## SERPINFO
### 1. Google Image
| Tab | Parameter | What gets detected |
|-----|-----------|--------------------|
| **All** | `udm=26` | Organic image results and inline image boxes |
| **Exact matches** | `udm=48` | Pages that contain the exact picture |
| **Visual matches** | `udm=44` | Visually similar pictures |


## Installation
1. Make sure the uBlacklist extension is installed and "Enable SERPINFO subscription links" is turned on (Options -> Advanced).
2. Click the subscription link above.
3. Confirm the subscription in the popup - that's it.
